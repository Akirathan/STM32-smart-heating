Detailed Description

This class controls heating.

Loads interval data from EEPROM in constructor and every minute checks if actual temperature corresponds to the temperature that was configured by the user in IntervalFrame.

Attention
controlTemperature method must be called to register for minute callbacks.
Note
MainFrame signals change in EEPROM data via reloadIntervalData method.
Implemented as a singleton class.

Checks if current temperature corresponds to the temperature that was configured by the user in interval setting.

If the temperature does not correspond, the heating (relay module) is switched on.

Note
Switches heating (relay module) off when no interval is active.
