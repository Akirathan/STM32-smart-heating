Detailed Description

Base class for SetIntervalFrame and OverviewIntervalFrame classes.

This frame consists of two time windows where user can set the interval time and one TempWindow when user can set temperature for that interval. This base class provides only "NEXT" and "END" buttons, SetIntervalFrame adds settable time and temp windows, whereas OverviewIntervalFrame adds non-settable windows.
